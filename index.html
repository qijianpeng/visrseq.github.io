VisRseq

R-based visual framework for analysis of sequencing data

View on GitHub

About

VisRseq, is a framework for analysis of sequencing datasets that provides a computationally rich and accessible framework for integrative and interactive analyses without requiring programming expertise.

Features

R-Apps

VisRseq provides a collection of R apps, with a auto generated and unified graphical user interface for popular computational packages in R and repositories such as Bioconductor.

Interactive Apps

To address the interactivity limitation inherent in R libraries, our framework include several native apps that provide exploration and brushing operations as well as integration with the IGV genome browser.
