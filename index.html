VisRseq, is a framework for analysis of sequencing datasets that provides a computationally rich and accessible framework for integrative and interactive analyses without requiring programming expertise.

Features

R Apps

Offering a semi-auto generated and unified graphical user interface for computational packages in R and repositories such as Bioconductor.

Interactive Apps

To address the interactivity limitation inherent in R libraries, our framework include several native apps that provide exploration and brushing operations as well as an integrated genome browser. The apps can be chained together to create more powerful analysis workflows.

Data manipulation

Creating data tables and processing sequencing data into data columns.

Computations on data columns

Using an excel or R syntax
