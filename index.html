--- layout: default title: VisRseq ---

VisRseq, is a framework for analysis of sequencing datasets that provides a computationally rich and accessible framework for integrative and interactive analyses without requiring programming expertise.

Features

  • R Apps: Offering a semi-auto generated and unified graphical user interface for computational packages in R and repositories such as Bioconductor.
  • Interactive Apps: several native apps provide exploration and brushing operations.
  • Chaining Apps: The apps can be chained together to create more powerful analysis workflows.
  • Built-in genome browser integration: Integrated with the Integrative Genome Viewer (IGV).
  • Data manipulation: Creating data tables and processing sequencing data into data columns.
  • Computations on data columns: Using an excel or R syntax.
